Avere — иметь

Урок 05 · Месяц 1

Месяц 1

«Иметь» вместо «быть»

По-русски
Мне холодно. Я голодный.
безличная конструкция или «я + прилагательное»
→
По-итальянски
Ho freddo. Ho fame.
буквально: «я имею холод / голод»

Многие физические ощущения, которые в русском выражаются через «мне + наречие» или «я + прилагательное», в итальянском передаются через avere + существительное. Это один из главных «ловушечных» паттернов для русскоговорящих.

Спряжение avere

ЛицоФормаПримечание
iohoh немой — читается просто «о»
tuhaih немой — читается «аи»
lui / leihah немой — читается «а»
noiabbiamo
voiavete
lorohannoh немой — читается «анно»
⚠ Внимание

Буква h в ho, hai, ha, hanno — немая. Она нужна только для отличия от омонимов: o (союз «или»), ai (предлог+артикль), a (предлог), anno (год). Никогда не произноси h!

Стандартное обладание

По-русски
У меня есть машина.
родительный падеж + «есть»
→
По-итальянски
Ho una macchina.
avere + прямое дополнение
Ho una macchina.
У меня есть машина.
Hai un cane?
У тебя есть собака?
Abbiamo un problema.
У нас есть проблема.

Устойчивые выражения с avere

Эти выражения нужно выучить как единое целое — они не переводятся дословно.

🇷🇺 Аналогия с русским

Запомни паттерн: там, где в русском «мне + наречие» (мне холодно, мне страшно, мне скучно), в итальянском ho + существительное (ho freddo, ho paura). Этот паттерн охватывает большинство физических и эмоциональных ощущений.

Возраст — тоже через avere

По-русски
Мне 28 лет.
без глагола «иметь»
→
По-итальянски
Ho 28 anni.
буквально: «я имею 28 лет»
Quanti anni hai?
Сколько тебе лет?
Ho ventotto anni.
Мне двадцать восемь лет.

Упражнение — переведи, используя avere

  1. Мне 25 лет.
  2. Ты голодна?
  3. Мы торопимся.
  4. У вас есть вопросы?
  5. Она боится.
Показать ответ
  1. Ho venticinque anni.
  2. Hai fame?
  3. Abbiamo fretta.
  4. Avete domande?
  5. Ha paura.
